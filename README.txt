COMO USAR O PROJETO PARA TRANSFERÊNCIA DE ARQUIVOS (De até 1 Megabyte):

-Primeiramente, execute o código do servidor pelo terminal, com o comando:
python3 server.py

-Em seguida, execute o código do cliente pelo terminal, especificando também o arquivo que você deseja que seja transferido (esse arquivo precisa estar no mesmo diretório do cliente), com o comando:
python3 client.py NOME_DO_ARQUIVO

Após isso, aguarde o fim da execução da transferência (não confiável, via UDP)
